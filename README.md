# Salary Prediction Using Machine Learning

## Project Overview
This project predicts employee salary based on years of experience using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Algorithm
- Linear Regression

## Features
- Load dataset
- Train model
- Predict salary
- Evaluate accuracy
- Visualize results

## Project Structure

Salary-Prediction/
│
├── data.csv
├── model.py

└── README.md

## Installation

pip install pandas matplotlib scikit-learn

## Run

python model.py

## Output
A graph showing Salary vs Experience with a Linear Regression line.
